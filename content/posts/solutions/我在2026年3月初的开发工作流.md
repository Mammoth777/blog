## 团队情况
目前我有3位编程勇士：
- github copilot: 收费套餐， claude opus 和 sonnet
- qwencode: auth 登陆的 1000次免费请求的qwen coder
- opencode: 基于 opencode zen 的免费 minimax-m2.5-free


## 工作流程

核心思路，copilot额度有限， 省着用，模型平台都是善人， 薅着用

### 场景A: 复杂功能设计， 架构设计
copilot opus完成计划， 产出doc和todo, 形成文件

### 场景B: 需求开发
- 简单需求： qwencode 或 opencode 上阵
- 复杂需求：claude sonnet

### 场景C: bug fix
- 优先： qwencode 或 opencode
- 前面俩搞不定的话，，claude sonnet
